# Independent project: Guestbook app

Vue frontend with Express backend using Axios to get/post form data.

Live demo: https://express-guestbook-app.herokuapp.com/

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
